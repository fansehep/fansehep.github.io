命令行工具分享

2023-01-24 Linux Command toolsRust PV:

引言

笔者在最近一年中接触到了很多命令行工具, 大多数都不错, 笔者有意将其分享出来.

彩色的 ls

ls 几乎可以说是你使用 Linux 最常用的命令之一, 普通的 ls 实在是不够让人看的舒服. exa 的彩色让人赏心悦目.

此外, 该命令还有额外的图标支持

github link: https://github.com/ogham/exa

btop vs htop vs btm

htop 大概应该足够好, 但 btop 似乎能给我们更多的体验和视觉?

btop github link: https://github.com/aristocratos/btop

btm 则似乎不讨喜.

更快的 grep

传统的 grep 的速度有时不那么如意, ripgrep 几乎完全兼容 grep, 而且速度有飞快一般的提升.

github link: https://github.com/BurntSushi/ripgrep

更好看的 cat

bat 可以给我们更好的视觉体验, 但他的复制效果是一场灾难. 此外 bat 可以和 manpage 一起使用.

github link: git@github.com:sharkdp/bat.git

更好看的查看空间命令

duf 用以取代 df, 更友好的查看当前设备空间状态

更好的 ps

procs 可以可以给出更友好的进程信息

mordern tree

dust 可以快速展示当前目录占用

终端的文件夹

mordern curl
